FAQ

O que é a In Visum?

A In Visum é uma plataforma de compartilhamento de bases de dados que pode ser utilizada via aplicativo mobile (android e iOS), web e API. Nosso intuito é o de criar um ponto de encontro digital no qual pesquisadores, desenvolvedores e outros cidadãos poderão submeter, visualizar e discutir dados dos mais diversos tipos --- desde informações dos portais de transparência do governo, até resultado de pesquisas acadêmicas..

Como usar o aplicativo?

O aplicativo foi construído para ser intuitivo e amigável ao usuário menos familiarizado com questões técnicas. Após o login, o usuário pode procurar bases de dados, adicioná-las para sua caixa de dados e, após selecionar todas que deseja utilizar, pode conferir as bases de dados, realizar operações simples e, finalmente, visualizar os dados. Uma alternativa é utilizar a opção "visualização rápida" (quando disponível) para visualizar rapidamente os dados selecionados.
